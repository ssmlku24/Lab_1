# Лабораторна робота: Метод Звичайного Жорданівського Виключення (ЗЖВ)

Цей репозиторій містить консольний додаток мовою C#, розроблений для виконання лабораторної роботи з лінійної алгебри / методів обчислень.

## 📌 Функціонал програми
Програма реалізує алгоритм ЗЖВ та дозволяє виконувати три основні операції:
1. **Пошук оберненої матриці** для квадратної матриці $n \times n$.
2. **Обчислення рангу матриці** довільного розміру $n \times m$.
3. **Розв'язання СЛАР** (Системи лінійних алгебраїчних рівнянь) за допомогою оберненої матриці ($X = A^{-1} \cdot B$).

## 💡 Особливості
- **Інтерактивне меню:** зручна навігація між завданнями.
- **Захист від помилок вводу:** програма перевіряє коректність введених розмірностей та елементів матриці (захист від вводу літер замість чисел або неправильної кількості елементів).
- **Детальне логування:** кожен крок обчислень (розв'язувальний елемент, стан матриці після кроку) виводиться на екран.
- **Експорт у файл:** результати роботи можна зберегти у текстовий файл (протокол) для використання у звітах.
