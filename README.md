# Лабораторна робота №1: Комплексний розв'язник задач лінійного програмування (ЗЛП)

## 📌 Опис проєкту
Цей проєкт є консольним додатком на C#, який реалізує повний алгоритмічний цикл розв'язання задач лінійного та цілочислового програмування. Він об'єднує класичні оптимізаційні алгоритми дослідження операцій, дозволяючи покроково знаходити оптимальні розв'язки для математичних моделей різної складності.

## 🚀 Реалізовані методи (Структура проєкту)

Проєкт розділено на чотири логічні частини (А, Б, С, Д), які викликаються з єдиного меню:

* **Частина А: Аналіз та побудова моделі** Програмно реалізовано алгоритм Звичайного Жорданівського Виключення (ЗЖВ). За допомогою розробленої програми було автоматизовано процеси знаходження оберненої матриці, обчислення рангу матриці та розв'язання систем лінійних алгебраїчних рівнянь матричним методом.
* **Частина Б: Класичний симплекс-метод** Розв'язання ЗЛП у канонічній формі (з обмеженнями типу `<=`). Програма автоматично додає балансові (додаткові) змінні, знаходить початковий опорний розв'язок і виконує кроки МЖВ до досягнення оптимуму.
* **Частина С: Узагальнений симплекс-метод (змішані обмеження)** Розширення симплекс-методу для роботи з будь-якими типами обмежень (`<=`, `>=`, `=`). Включає алгоритм пошуку допустимого розв'язку (адаптація двоїстого симплекс-методу) для виведення штучних змінних з базису.
* **Частина Д: Цілочислове програмування (Метод Гоморі)** Реалізація методу відтинаючих площин. Алгоритм знаходить неперервний оптимум, аналізує змінні на наявність дробових частин і автоматично генерує додаткові обмеження (відсікання Гоморі) до тих пір, поки не буде знайдено цілочисельний розв'язок.

## ⚙️ Особливості реалізації
* **Інтерактивне меню:** Зручний вибір потрібного методу для розв'язання задачі.
* **Захист від зациклень:** Обробка суперечливих систем обмежень та функцій, не обмежених зверху.
* **Детальне логування:** Вивід усіх проміжних симплекс-таблиць, що дозволяє використовувати програму як навчальний інструмент для перевірки розрахунків "на папері".
* **Збереження результатів:** Можливість експорту ходу розв'язання у текстовий файл.

## 🛠 Як запустити
1. Відкрийте рішення (Solution) у Visual Studio або сумісному IDE.
2. Скомпілюйте та запустіть проєкт.
3. У консольному меню оберіть потрібний режим (Частину) та введіть вхідні параметри задачі (кількість змінних, обмежень, коефіцієнти).
